# neon-budget-zen
Neon budget zen is a personal finance and budget management web application designed to help track expenses, manage budgets, and gain insights into their spending habit through a intuitive and user friendly interface
## features
- expense tracking
- spending analysis
- personal financial organization
- simple and intuitive dash board
